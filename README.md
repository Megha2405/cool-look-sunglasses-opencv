# cool-look-sunglasses-opencv
# Adding Sunglasses for a Cool New Look

This project uses image processing to add sunglasses to a face image, giving a simple and fun transformation.

## Overview
It demonstrates how to detect a face and place a sunglasses overlay using OpenCV. The project is beginner-friendly and useful for learning basic computer vision concepts.

## Features
- Detects face in an image  
- Adds sunglasses overlay on the eyes  
- Works with passport-size photos  
- Easy to customize with different glasses  

## Technologies Used
- Python  
- OpenCV  
- NumPy  

##  How to Use
1. Clone the repository  
2. Add your image to the `images` folder  
3. Run the script  
4. View your transformed image  

## 🎯 Applications
- Learning image processing basics  
- Simple computer vision practice  
- Fun photo editing
